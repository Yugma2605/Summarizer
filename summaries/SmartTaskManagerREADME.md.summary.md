Summary:
Smart Task Manager is a lightweight tool designed for developers and teams to organize tasks, set priorities, and track progress, featuring functionalities such as creating, updating, and deleting tasks, assigning tags and priority levels, tracking completion progress, and exporting tasks to JSON or CSV, with REST API support. It utilizes a tech stack including Python Flask for the backend, SQLite for the database, React for the frontend, and Docker for containerization.

Key Points:
- Organizes tasks, sets priorities, and tracks progress.
- Supports creating, updating, and deleting tasks.
- Allows assigning tags and priority levels to tasks.
- Enables exporting tasks to JSON or CSV formats.
- Offers REST API support for task management.
- Built with Python Flask, SQLite, React, and Docker.
